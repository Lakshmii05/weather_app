# 🌦️ Weather App

A simple Weather App built using HTML, CSS, and JavaScript. The application allows users to search for a city and view current weather information in an easy-to-use interface.

## Features

* Search weather by city name
* Display current temperature
* Show weather conditions
* Display humidity and wind speed
* Responsive and user-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## How to Use

1. Enter a city name in the search box.
2. Click the Search button.
3. View the current weather details for the selected city.

## Project Purpose

This project was created to practice frontend web development skills, including working with APIs, JavaScript, and responsive web design.
